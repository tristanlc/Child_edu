## 设计背景与理念

### 青春期孩子的特点

青春期是孩子成长的关键时期，这个阶段的孩子：
- 渴望独立和自主
- 对父母的管教容易产生抵触
- 自律能力仍在发展中
- 需要适当的激励机制

### 我们的设计原则

| 原则 | 说明 |
|------|------|
| 🌟 正向激励 | 用奖励代替惩罚，鼓励积极行为 |
| 🤝 平等尊重 | 让孩子参与规则制定，感受被尊重 |
| 💪 自主管理 | 培养自我监督能力，而非被动服从 |
| 🎯 循序渐进 | 从易到难，建立信心 |

## 积分机制

### 基础规则

```
📅 积分周期：每周一至周日
🎯 完成目标：当周任务的 85%
⏰ 游戏时间：达标可获得 1 小时/周
```

### 积分计算方式

| 完成率 | 获得游戏时间 | 说明 |
|--------|-------------|------|
| 85% 及以上 | 1 小时 | 达成目标 |
| 100% | 1 小时 + 表扬 | 超额完成，值得鼓励 |
| 85% 以下 | 0 小时 | 需要继续努力 |

### 周积分计算示例

假设本周任务总数为 20 项：
- 完成 17 项（85%）→ 获得游戏时间 ✅
- 完成 15 项（75%）→ 未达标 ❌

## 特赦令机制

### 什么是特赦令？

> 每月一次的"降低门槛"机会，帮助孩子在困难时期也能获得奖励

### 规则说明

| 项目 | 平时 | 特赦令期间 |
|------|------|-----------|
| 积分兑换门槛 | 85% 任务完成 | 70% 任务完成 |
| 游戏时间 | 1 小时 | **2 小时** |
| 有效期 | 1周 | 1次/月底 |

### 使用建议

```
⏰ 建议时间：每月最后一周
💡 使用场景：
   - 考试周（学习压力大）
   - 生病或特殊情况
   - 情绪低落需要鼓励
```

## 常见问题

### Q1：孩子抵触积分制度怎么办？

**建议**：
- 不要强制执行，把它当作"游戏"而非"任务"
- 让孩子参与规则制定，增加认同感
- 先从简单的任务开始，建立信心

### Q2：孩子作弊怎么办？

**建议**：
- 信任是基础，不要过度监控
- 可以和孩子约定"自检清单"
- 如果发现，鼓励诚实，比惩罚更重要

### Q3：孩子总是达不到85%怎么办？

**建议**：
- 适当调整任务数量或难度
- 使用"特赦令"给孩子鼓励
- 关注进步而非结果

### Q4：积分可以跨周累积吗？

**建议**：
- 可以设置"愿望积分池"
- 鼓励孩子为大奖存积分
- 设定合理的长期目标

### Q5：可以用积分抵消惩罚吗？

**不建议**：
- 积分制度应该正向激励
- 惩罚和奖励分开处理
- 避免"讨价还价"的心态
