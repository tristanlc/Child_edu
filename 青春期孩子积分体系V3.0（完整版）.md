# 🌟 青春期孩子积分体系 V3.0（完整版）

> **制定日期**：2026年4月  
> **适用对象**：青春期孩子（12-18岁）  
> **核心理念**：付出85%以上努力 → 每周保底可兑换1小时游戏

## 二、每日任务积分表

### A. 日常必选任务（每日至少选做4项）

| 任务类别 | 具体任务 | 积分 | 完成要求 |
|---------|---------|------|---------|
| **学习类** | 完成当日作业 | 8分 | 需家长签字确认 |
| | 课外阅读30分钟 | 6分 | 需做简要记录 |
| | 预习/复习课程 | 5分 | 记录学习内容 |
| | 额外练习/学习 | 8分 | 自主设定目标 |
| **家务类** | 收拾自己房间 | 5分 | 整洁达标 |
| | 饭后洗碗 | 4分 | 清洗干净 |
| | 帮忙做饭/准备食材 | 8分 | 实际参与 |
| | 倒垃圾/整理公共区域 | 3分 | 主动承担 |
| **运动类** | 户外运动30分钟 | 6分 | 需记录活动 |
| | 室内锻炼/跳绳1800/拉伸 | 4分 | 10分钟以上 |
| **成长类** | 情绪管理成功 | 10分 | ⭐当天无发脾气 |
| | 主动与父母沟通 | 15分 | ⭐⭐最鼓励！ |
| | 温和表达不满 | 8分 | 非暴力沟通 |

### B. 每日积分参考

```
假设每日完成5个任务：
   • 基础完成（4项）= 约30-35分
   • 优秀完成（5项）= 约40-50分
   
一周期望总积分：
   • 普通完成 = 210-245分（70-80%完成度）
   • 达标完成 = 245-280分（85%完成度）⭐
   • 优秀完成 = 280-350分（100%完成度）
```

## 十、给家长的话 💕

### 核心理念

这个制度的设计核心是：

1. **保底机制**：85%努力就能玩1小时，给孩子安全感
2. **特赦缓冲**：每月一次的轻松机会，避免僵化
3. **自主发声**：愿望清单让孩子有表达权，减少对抗
4. **阶梯目标**：付出越多收获越大，建立正确的价值观

### 沟通建议

> ✅ "这周我们一起努力，争取周末玩个痛快！"
> ✅ "你的愿望清单我已经看到了，我们一起想办法实现！"
> ✅ "这个月特赦令先留着，看看哪周特别需要的时候用！"

### 关于青春期沟通

青春期是孩子身心发展的关键时期，这个阶段的孩子：
- **渴望独立与被尊重**
- **情绪波动较大**
- **开始疏离父母但仍需要家庭支持**

**⚠️ 重要提醒**：和孩子沟通时，**语气永远比内容更重要**！

| ✅ 要这样做 | ❌ 不要这样做 |
|-----------|--------------|
| "我们一起看看怎么解决这个问题，好吗？" | "你怎么又做错了！" |
| "妈妈/爸爸理解你的感受，能说说吗？" | "你不要哭！有什么好哭的！" |
| "今天有什么想聊的吗？不强制哦～" | "你给我过来！" |
